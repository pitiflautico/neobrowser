Start in 30 seconds

$ git clone https://github.com/nichochar/neobrowser.git
$ cd neobrowser && cargo build --release

# Browse a page
$ ./target/release/neobrowser_rs see https://example.com

# As MCP server
$ ./target/release/neobrowser_rs mcp
