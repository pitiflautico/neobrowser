Inicio rapido

Funcionando en 60 segundos.

terminal
# Instalar
git clone https://github.com/pitiflautico/neobrowser.git
cd neobrowser && cargo build --release
# Navegar cualquier web
./target/release/neobrowser_rs see https://example.com
# Lanzar servidor MCP para tu IA
./target/release/neobrowser_rs mcp
