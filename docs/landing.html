Why does this exist?

AI agents need the web. But Chrome is 300MB, detectable, and slow. NeoRender is 15MB, invisible, and faster.

V8

Real JavaScript Engine

Same V8 that powers Chrome and Node.js. Executes React, Vue, Angular — any SPA. First to hydrate React Router 7 in headless V8.

TLS

Chrome 136 Fingerprint

HTTP requests are indistinguishable from real Chrome. JA3/JA4 hash, HTTP/2 settings, header order — all match. 19/20 top sites pass Cloudflare.

0px

No Pixels

No rendering pipeline. No layout. No paint. No GPU. The DOM exists in memory, JavaScript executes, but nothing is ever drawn. Pure computation.

How it's built

Three layers. No shortcuts.

HTTP Layer

rquest with Chrome 136 TLS emulation. BoringSSL. Full Sec-Ch-Ua, Sec-Fetch-* headers. Cookie jar in SQLite. HTTP cache on disk.

DOM Layer

linkedom provides the DOM API. Elements, attributes, events, selectors. Polyfills for ViewTransition, getAll, MutationObserver.

JS Layer

deno_core (V8) executes ES modules. On-demand fetching, code cache, module stubbing. 30 JS polyfill modules for browser APIs.

V8 Bytecode Cache

Compiled JavaScript cached to disk. Second load skips compilation. Same technique Chrome uses internally.

Module Stubbing

63 heavy modules (82MB) replaced with 200-byte Proxy stubs. Only essential modules fully parsed. Exports preserved via regex extraction.

Parallel Pre-fetch

362 modules fetched in parallel (depth 2). Disk-cached. Second load: 0.6s for all modules vs 6.5s from network.

Get Started

One binary. No dependencies except Rust toolchain to build.

$ git clone https://github.com/pitiflautico/neobrowser.git
$ cd neobrowser && cargo build --release
Compiling neobrowser_rs v0.9.0
Finished release [optimized] target(s) in 2m 30s

# As MCP server (Claude Code, Cursor, etc.)
$ echo '{"mcpServers":{"neobrowser":{"type":"stdio","command":"./target/release/neobrowser_rs","args":["mcp"]}}}' > ~/.claude.json

# Browse a page
$ ./target/release/neobrowser_rs see https://example.com

# Login to a site (saves cookies for future use)
$ ./target/release/neobrowser_rs login https://chatgpt.com
